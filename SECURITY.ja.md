## 脆弱性の報告

セキュリティ上の脆弱性を発見した場合は、**公開の Issue は作成しないでください**。以下のいずれかの方法で報告してください：

- **メール**: [セキュリティ用メールアドレスを記載]
- **GitHub Security Advisory**: このリポジトリの Security タブの「Report a vulnerability」ボタンを使用

### 報告に含める内容

- 脆弱性の説明
- 再現手順
- 想定される影響
- 修正案（あれば）

### 対応の目安

- 報告受領後 48 時間以内に確認の連絡
- 7 日以内に詳細な回答
- 進捗の共有

### 開示ポリシー

- 問題の理解と迅速な解決に協力します
- 希望に応じて発見者をクレジットします
- 修正が利用可能になった時点で公開開示を調整します
