# セキュリティポリシー

## サポート対象バージョン

現在、以下のバージョンにセキュリティ更新を提供しています。

| バージョン | サポート          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |
| < 0.1   | :x:                |

## 脆弱性の報告

セキュリティ上の脆弱性を発見した場合は、**公開の Issue は作成しないでください**。代わりに、以下のいずれかの方法で報告してください。

- **メール**: [セキュリティ用メールアドレスを記載]
- **GitHub Security Advisory**: このリポジトリの Security タブにある「Report a vulnerability」ボタンを使用する

### 報告に含める内容

脆弱性を報告する際は、以下を含めてください。

- 脆弱性の説明
- 再現手順
- 想定される影響
- 修正案（あれば）

### 対応の目安

- 報告を受領してから 48 時間以内に確認の連絡をします
- 7 日以内に詳細な回答を提供します
- 進捗について随時お知らせします

### 開示ポリシー

- 問題を理解し、迅速に解決するため、報告者と協力します
- 希望に応じて発見者をクレジットします
- 修正が利用可能になった時点で、公開開示のタイミングを調整します

i18next-turbo とその利用者の安全にご協力いただき、ありがとうございます。
